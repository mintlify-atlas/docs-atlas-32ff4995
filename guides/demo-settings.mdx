The package ships with a set of ready-made settings across several categories. These are intended as working examples and starting points — not production requirements.

<Info>
  Demo settings are fully functional but exist for illustration purposes. Remove any you do not need before shipping your project.
</Info>

## Using demo settings as templates

The demo classes follow the exact same structure you would use for your own settings. Copy a class, rename it, and update the key, default value, and metadata.

For example, to create a custom volume setting based on `MusicVolumeSetting`:

```csharp
[Metadata("Ambient Volume", "Adjust ambient sound volume",
    SettingCategory.Audio, "Volume", UIPreference.SliderStepper)]
public sealed class AmbientVolumeSetting : IntSetting
{
    public override string Key => "audio.ambient_volume";
    public override int DefaultValue => 40;
    public override int MinValue => 0;
    public override int MaxValue => 100;
}
```

See [Creating settings](/guides/creating-settings) for the full step-by-step walkthrough.
